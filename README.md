# Blog Muzyczny
Aby odpalic backend nalezy odpalic dockera z bazą mongoDB
aby mieć połączenie proszę wejść do terminala -> repo projektu i wpisać komendę
node baza_mongo/index.js

Aby wgrac przykladowe dane nalezy przy odpalonym backendzie wykonac komende node baza_mongo/import_danych.js

Na stronie możemy dodawać, edytować oraz usuwać utwory. To samo tyczy się również postów i komentarzy.
Dodatkowo po kliknięciu na pola Author lub Title w zakładce "MUZYKA" zostaną one posortowane kolejno wg autora lub tytułu utworu.
